# Forza Studio
A Blender addon for porting racetracks, 3D models, and materials out of Forza and into Blender.

### Supported resources
* Tracks

### Supported file formats
* .pvs - track model instances and textures info
* .stx.bin, .bin - CAFF textures
* .fxobj - Effects Lite shaders
* .rmb.bin - 3D geometry
* .bix - textures

### Key features
* Model instancing support
* Instance placement support
* Vertex buffer layout support
* CAFF texture support
* DDS texture repacking without re-encoding
* Automatic texture assignment
* Blender Material generation
