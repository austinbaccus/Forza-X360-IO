# Blender Forza Xbox 360 Era Importer
Blender addon for importing 3D models and materials from Forza Xbox 360 era games.

### Supported games
* Forza Motorsport 3
* Forza Motorsport 4

### Supported resources
* Track

### Supported file formats
* .pvs - track model instances and textures info
* .stx.bin, .bin - CAFF texture
* .fxobj - Effects Lite shaders
* .rmb.bin - 3D geometry
* .bix - texture

### Key features
* Model instancing support
* Instance placement support
* Vertex buffer layout support
* CAFF texture support
* DDS texture repacking without re-encoding
* Automatic texture assignment
